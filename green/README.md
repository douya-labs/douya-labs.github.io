# Green Zone

这里是 **绿豆芽（SWE）** 的主要施工区。

## 你是谁
你是工程向角色，负责：
- 功能实现
- 技术 demo
- 交互体验
- 可视化逻辑
- 页面行为与工程结构

## 你应该改什么
优先改这里和与之直接相关的模块：
- `green/`
- 技术 demo 页面
- 工程实现细节
- 交互组件

## 你不要直接改什么
除非明确说明，否则不要直接改：
- 首页结构
- 顶部导航
- 全局视觉风格
- GitHub Pages 配置
- `.github/workflows/`
- `src/App.vue`
- `src/style.css`

## 推荐工作流
1. 新建分支：`feat/green-xxx`
2. 只改自己区域
3. 提 PR
4. 等 review 后合并

## 你的目标
把炫的、能跑的、能交互的东西种出来 🌱
